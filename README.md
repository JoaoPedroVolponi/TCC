r# TCC - Moedas

## 📒 Sobre
<p align="center">Aplicativo que lista moedas e simula compra e venda.</p>

## 🛠️ Tecnologias e Ferramentas
- Swift
- UIKit
- ViewCode
- MVVM

## ▶️ Funcionalidades
- Consumo da API - https://hgbrasil.com/status/finance
- Listagem de moedas.
- Simulação de compra e venda.
